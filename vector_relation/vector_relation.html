ベクトルは「どこにあるか」を気にしない 📐

図形における矢印でベクトルを考えるとき、最も重要なルールのひとつが「平行移動して重なるベクトルは、すべて同じベクトルとみなす」ということです。

ベクトルの基本的な関係

  • 等しいベクトル ($\boldsymbol{a} = \boldsymbol{b}$)：
    大きさと向きが完全に同じベクトル。始点の位置が違っていても等しいとみなします。
  • 逆ベクトル ($\boldsymbol{a} = -\boldsymbol{b}$)：
    大きさは同じで、向きが正反対（180度逆）のベクトル。各成分の符号が逆になります。
  • 平行なベクトル ($\boldsymbol{a} // \boldsymbol{b}$)：
    向きが同じ、または正反対のベクトル。大きさが違っても平行です（$\boldsymbol{a} = k\boldsymbol{b}$ と表せます）。

各ベクトルの成分

$\boldsymbol{a} = ( \dots, \dots )$
$\boldsymbol{b} = ( \dots, \dots )$
$\boldsymbol{c} = ( \dots, \dots )$
$\boldsymbol{d} = ( \dots, \dots )$
