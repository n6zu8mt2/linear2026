内分・外分と線形結合の深い関係 📐

任意の点 $\vec{p}$ を2つの基底ベクトル $\vec{a}, \vec{b}$ を用いて $\vec{p} = s\vec{a} + t\vec{b}$ と表すとき、この $s$ と $t$ はこの斜交座標系における座標の役割を果たします。

内分・外分の公式を分解してみる

線分 $AB$ を $m:n$ に内分する点 $\vec{p}$ は、公式より $\vec{p} = \frac{n\vec{a} + m\vec{b}}{m+n}$ となります。これを分解すると：

$$\vec{p} = \left(\frac{n}{m+n}\right)\vec{a} + \left(\frac{m}{m+n}\right)\vec{b}$$

ここで $s = \frac{n}{m+n}, t = \frac{m}{m+n}$ と置くと、$\vec{p} = s\vec{a} + t\vec{b}$ という線形結合になります。 さらに、このとき $s+t = \frac{n+m}{m+n} = 1$ が常に成り立ちます。

外分の場合も同様に $s+t=1$ となり、「係数の和が1になる」という条件が、「2点を通る直線上にある」ことと同値であることがわかります。
